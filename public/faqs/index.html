Modular

Hide and show some elements easily from the config file.

Multilingual

The default content language is english (it can be changed), but you can add as many languages as you want very easily, the theme includes an italian language example and a dropdown men√π for easily switching between languages. Permalinks can be localized in each language (eg. you can have `/contacts` for english and `/contatti` for italian).

Customizable

The page content can be customized from within the config files.

SEO ready

Every page in /content has an easily customizable meta description and a unique meta title for better and easier indexing. The theme will use the image `static/img/meta-image` as meta image for Google, Meta and X.

Google Analytics supported

Insert your Google Analytics tag in the `config.toml` file and be ready to track your site performances easily.

Google Maps Included

The contact page can be easily customized by adding a google maps link to get directions towards your activity location.
